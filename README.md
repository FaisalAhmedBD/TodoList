# TodoList
### Used Technologies
1. Backend
   - Express.js, MongoDB, mongoose
2. Frontend
   - Reactjs, bootstrap
##  Features
      - Add new task
      - Edit a task
      - Mark a task as completed
      - Delete a task
      - Clean the whole task list
# Build instruction
1.  clone the repo
3.  move to Backend and Frontend sub-folder and run `npm install`
4.  make sure that your mongodb instance is running
